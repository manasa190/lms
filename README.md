# Learnova - Learning Management System

Learnova is a modern, feature-rich Learning Management System designed to provide a seamless learning experience for students and a powerful management interface for instructors and admins.

## Features

- **Course Management**: Create and manage detailed courses with modules and lessons.
- **Video Learning**: Deep integration with YouTube for high-quality video lessons.
- **Interactive Quizzes**: Test your knowledge with built-in quizzes.
- **Gamification**: Earn points, track progress, and climb the leaderboard.
- **Student Dashboard**: Personalized insights and recommendations.
- **Admin Tools**: Comprehensive user and analytics management.

## Tech Stack

- **Frontend**: Vite, React, TypeScript
- **Styling**: Tailwind CSS, shadcn-ui
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Data Visualization**: Recharts

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Development

- `npm run dev`: Start development server
- `npm run build`: Build for production
- `npm run lint`: Run linting checks
- `npm run test`: Run unit tests
